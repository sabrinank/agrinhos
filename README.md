## 📋 Sobre o Projeto

Este repositório contém o código-fonte de um website moderno, responsivo e focado na consciencialização sobre a importância do **Programa Agrinho 2026**. O objetivo principal da página é demonstrar como a tecnologia digital (como a inteligência artificial, robótica e desenvolvimento web) pode ser uma forte aliada da sustentabilidade no agronegócio, estreitando os laços entre o meio rural e o urbano.

O site foi desenvolvido utilizando as melhores práticas de semântica e estilização para garantir uma navegação fluida tanto em computadores como em dispositivos móveis.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído do zero, sem a utilização de frameworks pesados, focando no desempenho e na pureza do código:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Design responsivo, estilização moderna com variáveis CSS e efeitos de transição suave.
* **Google Fonts:** Tipografia focada na legibilidade (*Poppins*).

---

## 💻 Estrutura de Páginas e Secções

O website está organizado numa arquitetura de página única (*Single Page Application* com navegação interna) que inclui:

1.  **Cabeçalho (Navbar):** Menu de navegação fixo e responsivo para fácil acesso às secções.
2.  **Hero Banner:** Apresentação de forte impacto visual com o lema do projeto e chamada para ação (*Call to Action*).
3.  **Sobre o Agrinho:** Secção informativa detalhando os pilares do programa (Inovação, Sustentabilidade e Cidadania) dispostos em formato de cartões (*cards*).
4.  **Importância e Destaque:** Secção focada na reflexão sobre o papel da tecnologia no campo moderno, acompanhada de elementos gráficos decorativos.
5.  **Rodapé (Footer):** Informações de direitos de autor e identificação da equipa/concurso.

## 🛠️ Como Executar o Projeto Localmente

Para abrir e visualizar o projeto no teu computador, não precisas de instalar nenhum software complexo. Basta seguires estes passos:

1.  **Clona este repositório:**
    ```bash
    git clone [https://github.com/](https://github.com/)[teu-usuario-do-github]/[nome-do-repositorio].git
    ```
2.  **Navega até à pasta do projeto:**
    ```bash
    cd [nome-do-repositorio]
    ```
3.  **Abre o site:**
    Basta dares um duplo clique no ficheiro `index.html` e ele abrirá automaticamente no teu navegador de internet (Chrome, Edge, Firefox, etc.).
